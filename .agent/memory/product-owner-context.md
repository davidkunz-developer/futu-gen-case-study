## Rozpracované úkoly (Zadání)
- Modul 1: Audio Streamer (Příjem a normalizace audio dat).
- Modul 2: Transcription Engine (Zpracování pomocí Whisper na JSON záznamy s konfidenčními scores a timestampy).
- Modul 3: Classifier (Rozlišení private / topic-based konverzací a témata).
- Bonusy: WebView / WebSocket stream, Docker Compose, Incremental klasifikace, Eval sada, Diarizace.
- Stack: Python (asyncio) / TS (Node.js 18+).
